# 📊 USA Superstore Dashboard

## 📌 Overview

The **USA Superstore Dashboard** is an interactive **Power BI** project built to analyze business performance using the USA Superstore dataset from **2014 to 2017**.  
It provides clear and actionable insights into:

- Sales performance
- Profitability
- Customer orders
- Shipping efficiency
- Return behavior
- Regional and manager performance

---

## Project Objectives

- Monitor overall business KPIs
- Analyze sales and profit trends over time
- Identify top-performing categories, sub-categories, cities, and regions
- Evaluate shipping performance and delivery delays
- Measure return rates and their business impact
- Support data-driven decision-making through interactive visuals

---
